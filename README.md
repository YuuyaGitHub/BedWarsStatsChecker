# Hypixel Bed Wars Stats Viewer
This is a C# console app that uses the [Hypixel API](https://api.hypixel.net/) to retrieve Bed Wars (Solo) statistics.
<br>You will need a Hypixel API key to use this program, see [here](https://www.google.com/search?q=how+to+get+hypixel+api+key) for instructions on how to get one.

# System Requirements
* [.NET 8](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
* Windows, macOS, Linux

# Libraries used
* [Newtonsoft.Json](https://www.newtonsoft.com/json)

# How to Build
1. Clone or download this repository as a ZIP file
1. Create an `api_key.txt` file in the same folder as the executable file
1. Enter the API key obtained from the Hypixel Developer Dashboard
1. Build and run
> [!IMPORTANT]
> The file name (`api_key.txt`) that stores the API key is hard-coded in the program and cannot be changed.
