# Hypixel BedWars Stats Viewer
This is a C# console app that uses the Hypixel API to retrieve BedWars (Solo) statistics.
<br>To use this program, you need to obtain a Hypixel API key. Please see the Wiki for instructions on how to obtain one.

# System Requirements
* [.NET 8](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
* Windows, macOS, Linux

# Libraries used
* [Newtonsoft.Json](https://www.newtonsoft.com/json)
